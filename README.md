# Tony – Cyber Security Analyst Portfolio

A modern, responsive cybersecurity portfolio website showcasing skills, projects, certifications, and experience in ethical hacking, penetration testing, and network security.

## 🎯 Overview

This is a professional MVP (Minimum Viable Product) portfolio designed for cybersecurity students, analysts, and ethical hackers seeking internship or attachment opportunities. The site features a dark cybersecurity theme with green terminal-style accents, smooth animations, and interactive elements.

## ✨ Features

- **Hero Section** – Animated glitch effect on name, typing subtitle effect, dual CTA buttons, and animated stat counters
- **About Me** – Terminal-style window with professional bio, quick-info cards, and skill tags
- **Technical Skills** – Categorized skill bars (Offensive Security, Network & Recon, Systems & Tools) with animated progress indicators
- **Projects** – Showcase of 4 cybersecurity projects with descriptions, tools used, and status badges
- **Certifications** – Display of completed and in-progress cybersecurity certifications and training
- **Contact** – Email, GitHub, LinkedIn contact cards plus a functional contact form
- **Responsive Design** – Fully mobile-friendly with hamburger navigation
- **Smooth Animations** – Scroll-triggered reveals, matrix rain background, cursor trail effects, and more

## 🛠️ Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Dark theme with CSS variables, animations, and responsive design
- **Vanilla JavaScript** – No frameworks; lightweight interactions and effects
- **Font Awesome 6.5** – Icon library
- **Google Fonts** – Cybersecurity-themed fonts (Orbitron, Rajdhani, Share Tech Mono)

## 🎨 Customization

### Update Personal Information

1. **Name & Title** – Edit the hero section in `index.html`
2. **About Me** – Update the terminal window content in the About section
3. **Contact Info** – Replace placeholder emails and social links:
   - Search for `tony@example.com` → your email
   - Search for `github.com/tony` → your GitHub profile
   - Search for `linkedin.com/in/tony` → your LinkedIn profile

### Modify Skills & Projects

- **Skills** – Update the skill bars in the Skills section with your proficiencies
- **Projects** – Add or edit project cards with your own projects and descriptions
- **Certifications** – Update the certification cards with your actual training and credentials

### Theme Customization

All colors are defined as CSS variables in `assets/css/style.css`:

```css
--accent:    #00ff88;        /* Green accent */
--blue:      #0080ff;        /* Blue accent */
--bg-primary: #050a0f;       /* Dark background */
```

Modify these to change the entire theme.

## 🌐 Deployment

This portfolio is deployed using **GitHub Pages**:

1. Push code to GitHub repository
2. Enable GitHub Pages in repository settings (Settings → Pages → Source: main branch)
3. Your site will be live at `https://[username].github.io/[repo-name]`

### Custom Domain (Optional)

To use a custom domain (e.g., `tony-portfolio.com`):

1. Purchase a domain from a registrar (GoDaddy, Namecheap, etc.)
2. Add a `CNAME` file to the repository root with your domain name
3. Update DNS records to point to GitHub Pages
